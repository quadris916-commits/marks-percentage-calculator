# marks-percentage-calculator
A simple Python Tkinter GUI project that calculates total marks, average, percentage, grade, and pass/fail status for 5 subjects.
## Features

- Enter marks for 5 subjects
- Calculates total marks
- Calculates average marks
- Calculates percentage
- Displays grade
- Displays pass/fail status
- Clear button to reset all fields

## Tech Used

- Python
- Tkinter

## Project Structure

marks-percentage-calculator/
├── marks_percentage_calculator.py

## How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Open terminal in the project folder.
4. Run:

```bash
python marks_percentage_calculator.py
```

## Grade Logic

- 90 and above = A+
- 80 to 89 = A
- 70 to 79 = B
- 60 to 69 = C
- 50 to 59 = D
- Below 50 = F
